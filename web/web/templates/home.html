About this project

This web application, plus its back-end components, are part of my final project for CU Boulder's "Applications of Software Architecture for Big Data" course.

The project showcases social media (and user submitted) posts about Vancouver, that have been analyzed for sentiment (Positive, Neutral, or Negative).

Features include:

  • Data collection from social media (through Bluesky's API) and user submissions
  • Data analysis via a sentiment analysis library
  • User submission via forms
  • This web interface to view all posts and their sentiment
  • Communication between components is handled via REST APIs and message queues
  • A data store: messages are stored in a PostgreSQL database for persistence
  • Instrumentation and monitoring for performance and reliability
  • Continuous integration and deployment using GitHub Actions
  • Production environment on Azure with Docker containers

How it works:

  1. You submit a post about Vancouver
  2. The system analyzes the sentiment (Positive, Neutral, or Negative)
  3. Results are added to the database and displayed in the table
